# IITM BS Support

## Description
IITM BS Support is a student-built web platform that provides academic tools for students enrolled in the IIT Madras BS program. It helps students calculate CGPA, convert marks to grades, and predict end-term performance.

## Features
- CGPA calculator based on IIT Madras credit structure
- Grade calculator using official grading scale
- End-term marks predictor
- Clean, responsive, and mobile-friendly UI
- No login and no data collection

## Tech Stack
- HTML
- CSS
- JavaScript

## Available Tools
- CGPA Calculator
- Grade Calculator
- End-Term Marks Predictor

## Project Structure
- `css/` – Stylesheets  
- `js/` – JavaScript logic  
- `pages/` – Informational pages (About, Privacy, Disclaimer)  
- `tools/` – Individual tool pages  
- `index.html` – Home page  
- `tools.html` – Tools listing page  

## How to Run Locally
1. Clone or download the repository
2. Open `index.html` in any modern web browser
3. Navigate using the menu to access tools
