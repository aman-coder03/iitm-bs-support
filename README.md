# IITM BS Support

**IITM BS Support** is a student-driven, browser-based academic assistance platform created for learners enrolled in the **IIT Madras Bachelor of Science (BS) Programme**.

The platform provides a suite of essential academic utilities that help students accurately calculate CGPA, interpret grades, and estimate examination performance. It is intentionally designed to be lightweight, fully client-side, and privacy-focused â€” requiring no login, backend services, or data storage.

Students enrolled in the IIT Madras BS programme frequently require dependable tools to:
- Calculate CGPA across Foundation, Diploma, and Degree levels
- Convert numerical marks into grades using official grading policies

Manual evaluation methods are often time-consuming and prone to inaccuracies.  
**IITM BS Support** addresses these challenges by delivering fast, accurate, and easily accessible academic tools within a unified interface.
